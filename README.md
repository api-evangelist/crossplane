# Crossplane (crossplane)
Open source Kubernetes add-on that transforms your cluster into a universal control plane, enabling you to manage cloud infrastructure, services, and applications using Kubernetes-style declarative configuration. Crossplane is a graduated Cloud Native Computing Foundation project.

## APIs

### Crossplane Kubernetes API
The Crossplane Kubernetes API extends the Kubernetes API with custom resources including Compositions, CompositeResourceDefinitions (XRDs), Providers, ProviderConfigs, and Claims. These resources allow declarative management of cloud infrastructure and services through standard Kubernetes manifests and the kubectl CLI.
